This woocommerce plugin allows the user to get payments through an API done with TusPagos.

== Description ==

TusPagos Custom Payment Gateway is a woocommerce plugin that allows you to integrate a payment gateway developed by TusPagos. It has multiple payment options, such as:
credit card payments, Visa or MasterCard Debit Cards, Venezuelan Pago Movil and Venezuelan Transfers. It integrates with WooCommerce

== Frequently Asked Questions ==

= How to get started? =

Generate API Key and encryption keys inside TusPagos dashboard to then be added in the "TusPagos Payment Gateway Option", then activate the payments inside the Woocommerce configuration

= How much does it cost? =

Tus Pagos has a porcentual fee for every transaction, which is a contribution for the development team to be able to maintain and support all of the plugin's updates. Also, this plugin has a trial period of 30 days in which your transactions would be totally free!
